在标头 <bit> 定义
template< class T >
constexpr T bit_floor( T x ) noexcept;
 (C++20 起)

若 x 非零，则计算不大于 x 的最大的二的整数次幂。若 x 为零，则返回零。

此重载只有在 T 为无符号整数类型（即 unsigned char、unsigned short、unsigned int、unsigned long、unsigned long long 或扩展无符号整数类型）时才会参与重载决议。

参数

x - 无符号整数值

返回值

若 x 为零则为零；否则为不大于 x 的最大的二的整数次幂。

注解

P1956R1 以前，为这个函数模板提出的名字是 floor2。

功能特性测试宏 值 标准 功能特性
__cpp_lib_int_pow2 202002L (C++20) 2 的整数次幂运算

可能的实现

template<typename T, typename ... U>
concept neither = (!std::same_as<T, U> && ...);
 
template<std::unsigned_integral T>
    requires neither<T, bool, char, char8_t, char16_t, char32_t, wchar_t>
constexpr T bit_floor(T x) noexcept
{
    if (x != 0)
        return T{1} << (std::bit_width(x) - 1);
    return 0;
}

示例

运行此代码
#include <bit>
#include <bitset>
#include <iostream>
 
int main()
{
    using bin = std::bitset<8>;
 
    for (unsigned x{}; x != 012; ++x)
        std::cout << "bit_floor( " << bin(x) << " ) = "
                  << bin(std::bit_floor(x)) << '\n';
}
