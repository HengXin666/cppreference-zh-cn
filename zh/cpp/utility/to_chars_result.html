在标头 <charconv> 定义
struct to_chars_result;
 (C++17 起)

std::to_chars_result 是 std::to_chars 的返回类型。它没有基类，并且只有以下成员。

数据成员

成员名字 定义
ptr
 char* 类型的指针
(公开成员对象)
ec
 std::errc 类型的错误码
(公开成员对象)

成员和友元函数

operator==(std::to_chars_result)

friend bool operator==( const to_chars_result&,
                        const to_chars_result& ) = default;
 (C++20 起)

通过默认比较（它使用 operator== 分别比较 ptr 和 ec）比较两个实参。

此函数对常规的无限定或有限定查找不可见，而只能在 std::to_chars_result 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

operator bool

constexpr explicit operator bool() const noexcept;
 (C++26 起)

检查转换是否成功。返回 ec == std::errc{}。
